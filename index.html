How Does Javascript Work

Javascript also reffered as ECMAScript is a single threaded, interpreted language. Interpreted means it is not a compiled language. Javascript by default run in browser directly using a Javascript Engine like Google Chrome uses V8, mozilla use SpiderMonkey etc. And most of the browser support Javascript.
As javascript is a single threaded language it handles one task at a time. And as a non compiled programming language Javascript use a modern technique called Just-In-Time Compilation or in short JIT compilation. in which Javascript execute its code into it

Difference Between Local Storage and Session Storage

The main difference in Local storage and session storage is that the data in sessionStorage is clear when the session ends. But the Local Storage not been clear when the session ends.
The storage capacity is between 5-10 MB. And the data should be deleted manually. The data not shared with server and client can only read the local storage.
In Session Storage everytime a session is been once a tab is loaded. Page session is only valid for a tab. Different tab that is open using the same url create its own session storage. Methods used in Storage:

  1. setItem()
  2. getItem()
  3. removeItem()
